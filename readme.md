#picture-this
image placeholder i php, en enda fil
        hjälp
        bredd
        höjd
        filformat (defaultad)
        fg-färg (=fff)
        bg-färg (=gray)
        text (=null)
        border (=0)
        cache (=1)
        ====
        argument via get eller argv
        så få dependencies som möjligt, gd eller imagick
        visar storleken i bilden
        liten storlek > snygg > allt annat
